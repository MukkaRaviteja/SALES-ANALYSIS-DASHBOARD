📊 Sales Dashboard (Excel Project)

🔹 Overview

This project is an interactive Excel dashboard used to analyze sales performance.

🔹 Features

* KPI Metrics (Total Sales, Profit, Quantity)
* Pivot Tables & Pivot Charts
* Slicers for filtering (Region, Category)
* Dynamic data visualization

🔹 Tools Used

* Microsoft Excel
* Pivot Tables
* VLOOKUP
* Conditional Formatting

🔹 Insights

* Identified top-performing regions
* Analyzed monthly sales trends
* Improved decision-making using visual data
